# autodns
Auto DNS is a tool for change the DNS servers in your dns configuration. Just choose the best dns provider for you, and be happy!

# usage
1 - cd into 'Auto\ DNS' folder.
2 - sudo python3 auto_dns.py
3 - Choose a dns server
4 - Ready.
